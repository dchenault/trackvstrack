# **App Name**: Track vs Track

## Core Features:

- Group Creation: Allow a user to create a group with a permanent name and generate a unique, shareable link.
- Group Joining: Enable users to join a group via a shared link and enter a display name.
- Album Creation: Allow the group owner to create an album by pasting a Spotify album URL; extract artwork and track list.
- Bracket Generation: Automatically generate a randomized, tournament-style bracket from the track list.
- Song Matchup & Voting: Present a voting window with two songs facing off; allow users to listen on Spotify and cast a vote.
- Live Vote Updates: Update vote counts live for all users and advance the winning song.
- Album Rating & Archiving: After bracket completion, prompt users to rate the album and archive it with ranking; let users access archived album data and bracket results.

## Style Guidelines:

- Primary color: Neon blue (#00FFFF) to reflect musical creativity and collaboration.
- Background color: Neon pink (#FF69B4), providing a subtle base to let the content stand out.
- Accent color: Neon purple (#D070FF) to complement the primary color.
- Body and headline font: 'Inter', a grotesque-style sans-serif, to achieve a modern, objective feel.
- The album bracket should take a central position.
- The Spotify integration must be intuitive and obvious.
- Custom icons to represent votes, rankings, and Spotify links.
- Use subtle transition animations between voting rounds.
